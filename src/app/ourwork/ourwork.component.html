Our Work

Our work is to develop the code solution to modify the google map based on the backend application data. We use a real time back-end data which contains water resources information. This data can be replaced with the actual real time data provided by real time water resource data API provider. We developed the back-end code which supplies data to front end. Front end makes necessary changes and displays the sample data on the google Map. Front end locates the user on google map based on geolocation and provides the water resource location. Nearby Water Resources information can be sent to user mobile for reference.

Front-End Features

  • Developed using Angular.
  • Locates user on google map.
  • Modify the google map based on the data provided by back end.
  • Suitable for mobile devices.
  • Progressive web app.
  • Deployed on IBM cloud as cloud foundry application.

Back-End Features

  • Developed using NodeJS.
  • Supplies real time water resource data to Front-end.
  • Perform send mobile message functionality.
  • Deployed on IBM cloud as cloud foundry application.
